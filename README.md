#### 活動介紹
我們與 R，並不遙遠！只需要連續七天跟著 R-Ladies Taipei 進入精神時光屋一起破任務拿龍珠，七天後擁有七顆龍珠的妳就可以跟我們一起尋找神龍！
很適合「攻城師」不敵壞人時使用，在精神時間屋中的七天，每天花 1-2 小時的 Kaggle 實戰修煉，妳也能成為資料分析女賽亞人！
